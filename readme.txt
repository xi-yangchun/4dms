4次元マインスイーパー

遊び方
・フィールドが4次元空間であること以外は、ふつうのマインスイーパーと同じです。
・爆弾のないマスを全て掘るとゲームクリアになります。

操作
・ゲームの起動：4dminesweeper.exeをダブルクリック
・マスの上で左クリック：マス上を掘る
・マスの上で右クリック：マス上に旗を立てる

4次元空間の見かた
・掘られたマスの数字：隣接している爆弾の個数

・4*4マスの小さな平面が4*4個並んで、4次元空間になっています。
　example0.pngを開いて見てみましょう。

・example1.pngを開いて見てみましょう。
　例えば☆印のマスは、赤枠の領域内の全てのマス(80個)と隣接しています。

・example2.pngを開いて見てみましょう。　
　例えば☆印のマスは、赤枠の領域内の全てのマス(53個)と隣接しています。

・example3.pngを開いて見てみましょう。　
　例えば☆印のマスは、赤枠の領域内の全てのマス(23個)と隣接しています。

注意点
・×ボタンの反応が悪く、なかなか終了しないことがあります。
　×ボタンを何回か押す、長押しするなどすると終了する場合があります。
